## Step Results

### Step 1 — Create FAQ Scraper

**Status:** Complete

- Created `scraper/faq_scraper.py` targeting Bootstrap accordion structure (`pxp-faq-accordion`, `card-header`, `card-body`).
- Filters by known FAQ categories to exclude non-FAQ cards (e.g., cookie consent).
- Generates deterministic IDs from category + question slug + MD5 hash.
- Successfully extracted **8 FAQ records** across 3 categories:
  - Ordering Products: 6 questions
  - Shipping & Delivery: 1 question
  - Become a Beauty Consultant: 1 question
- Output: `scraper/faq_records.jsonl`

**File created:** `scraper/faq_scraper.py`

### Step 2 — Create FAQ Loader (Embed + Upload to Azure AI Search)

**Status:** Complete

- Created `scraper/faq_loader.py` (uses `search-site/.venv` for Azure SDK).
- Added `page_type` field (String, filterable) to the `marykay-products` index schema — field did not previously exist.
- Generated 3072-dim embeddings for all 8 FAQ records via `text-embedding-3-large`.
- Mapped FAQ fields to index fields: `question` → `title`/`h1`, `answer` → `main_text`, `content` → `description`, empty strings for product-specific fields.
- Uploaded via `merge_or_upload_documents()`: **8 succeeded, 0 failed**.

**File created:** `scraper/faq_loader.py`
